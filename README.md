# 🌤️ Weather App

A simple weather app built using HTML, CSS, and JavaScript as part of my FY IT learning journey.

---

## 🚀 Features
- Search any city in the world
- Shows current temperature in °C
- Shows weather condition with emoji (☀️ ⛅ 🌧️ ❄️)
- Shows humidity and wind speed
- Shows error message if city is not found

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

---

## 📚 What I Learned
- How to use a **real API** (OpenWeatherMap) to get live weather data
- How to use **fetch()** to send a request to an API
- How to use **async / await** to handle API response
- How to use **JSON data** returned from the API and display it on screen
- How to connect **JavaScript with HTML** using `getElementById` and `innerHTML`
- How to add **event listeners** for button click and Enter key
- How to use **try / catch** to handle errors properly
- How to style a webpage using **CSS** (gradient background, flexbox, border radius)
